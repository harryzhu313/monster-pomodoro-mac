# Security Policy

## 报告漏洞 / Reporting a Vulnerability

如果你在本项目中发现安全漏洞，请**不要**通过公开 Issue 报告。

请使用 GitHub 的 [Private Vulnerability Reporting](../../security/advisories/new) 私下提交，我会尽快回应。

If you discover a security vulnerability, please **do not** open a public issue.
Use GitHub's Private Vulnerability Reporting (Security tab → "Report a vulnerability") instead.

## 范围说明

- 本应用为本地优先（local-first）的 macOS 菜单栏番茄钟，不运行任何服务端。
- Notion token 等凭据仅存储在用户本机的 app data 目录，仓库中不应出现任何真实密钥。若你在仓库或发布产物中发现疑似泄露的凭据，请立即通过上述渠道告知。
